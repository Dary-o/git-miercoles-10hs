REMOTE REPOSITORY
-----------------

		"Es un repositorio alojado en un servidor u otra computadora 
		distinta a la ques estamos utilizando"

COMANDOS
		"como todos los comandos de GIT, estan precedidos por dicha
		palabra y estos en particular, nos van a pedir agregar
		repositorios, eliminarlos, etc."

LISTA DE COMANDOS:

git remote       " lista de los repositorios remotos"
git remote -v    "muestra las direcciones de los repositorios remotos"
	git remote add " ALIAS" <url> => " agrega un repositorio remoto"
	git remote remove "ALIAS"
